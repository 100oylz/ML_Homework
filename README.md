# 机器学习作业

## 数据集

### 1. 零售销售预测 (Sales Forecasting)

| 属性 (Attribute)  | 详情 (Details)                                               |
| ----------------- | ------------------------------------------------------------ |
| **数据集 Handle** | `rohitsahoo/sales-forecasting`                               |
| **主题领域**      | 零售和商业分析                                               |
| **数据粒度**      | 通常为**日级**或**周级**。                                   |
| **数据范围**      | 包含多个商店和商品的销售数据，跨度可能为数年。               |
| **主要特征**      | 包含日期、**商店/商品ID**、销售数量、价格，有时可能包含促销、节假日等**协变量 (Exogenous Variables)**。 |
| **时序特点**      | 具有强烈的**季节性**（如年末假期、购物节）和**周期性**（如每周的销售高峰）。趋势受宏观经济影响。 |
| **建议用途**      | 经典的预测任务，适合测试模型处理**多变量输入**和捕捉**复杂非线性季节性**的能力。可作为对比 MLP、LSTM 和 Transformer 性能的理想选择。 |

### 2. 电力消耗预测 (Electric Power Consumption)

| 属性 (Attribute)  | 详情 (Details)                                               |
| ----------------- | ------------------------------------------------------------ |
| **数据集 Handle** | `fedesoriano/electric-power-consumption`                     |
| **主题领域**      | 能源和负荷预测                                               |
| **数据粒度**      | 通常为**月级**或**小时级**（取决于具体文件）。该 Handle 主导文件为月度生产指数。 |
| **数据范围**      | 记录了某地区或电网的电力生产或消耗指标，时间跨度通常较长。   |
| **主要特征**      | **单变量时间序列**。核心特征是随时间变化的电力生产/消耗数值。 |
| **时序特点**      | 具有非常稳定和明显的**年度季节性**（例如，夏季空调和冬季取暖导致的高峰）以及明确的**长期增长趋势**。序列相对平滑、噪声较小。 |
| **建议用途**      | **适合作为基础模型 (作业 1)** 的入门数据集，用于清晰地展示 MLP、CNN、LSTM 如何识别和拟合周期性和趋势。也适合用于**长序列预测**的基准测试。 |

### 3. Yahoo 股票价格预测 (Yahoo Stock Price)

| 属性 (Attribute)  | 详情 (Details)                                               |
| ----------------- | ------------------------------------------------------------ |
| **数据集 Handle** | `arashnic/time-series-forecasting-with-yahoo-stock-price`    |
| **主题领域**      | 金融、股票市场分析                                           |
| **数据粒度**      | **日级**数据。                                               |
| **数据范围**      | 包含多个知名公司股票、ETF 或指数的历史价格数据。             |
| **主要特征**      | **多变量时间序列 (Multi-variate)**。包含 OHLCV（开盘价、最高价、最低价、收盘价、交易量）等金融指标。 |
| **时序特点**      | **高波动性、高噪声、非平稳性**。趋势和变化受市场情绪、突发新闻、宏观经济等大量**外部因素**影响，难以预测。 |
| **建议用途**      | **挑战性最高的任务**，非常适合在 **作业 2** 中使用 **Transformer 模块**来捕捉复杂的、非线性的、长期的市场依赖关系，并探索多变量输入的建模效果。 |

### 4. 日常气候时间序列数据 (Daily Climate Time Series)

| 属性 (Attribute)  | 详情 (Details)                                               |
| ----------------- | ------------------------------------------------------------ |
| **数据集 Handle** | `sumanthvrao/daily-climate-time-series-data`                 |
| **主题领域**      | 气象与环境预测                                               |
| **数据粒度**      | **日级**数据。                                               |
| **数据范围**      | 某地区多年的每日气候记录，如印度德里等城市的气候数据。       |
| **主要特征**      | **多变量时间序列**。包含 `temperature` (温度)、`humidity` (湿度)、`wind_speed` (风速)、`mean_pressure` (平均气压) 等多个气象变量。 |
| **时序特点**      | 具有明显的**年度周期性**，但天气变化具有高度的**随机性**和**非线性相关性**。不同特征之间存在明显的物理关联（如气压和风速）。 |
| **建议用途**      | 适合进行**多输入多输出 (MIMO)** 或 **多输入单输出 (MISO)** 预测任务。可以利用多个气象协变量来提高单一变量的预测精度。是测试 **Transformer/Attention** 在处理多维特征序列上性能的优秀数据集。 |
